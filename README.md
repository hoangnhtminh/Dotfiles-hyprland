## 📦 Stack

| Component | Package |
|-----------|---------|
| WM | [Hyprland](https://hyprland.org) |
| Bar | [Waybar](https://github.com/Alexays/Waybar) |
| Launcher | [Rofi](https://github.com/davatorium/rofi) |
| Terminal | [Kitty](https://sw.kovidgoyal.net/kitty) |
| Notification | [Swaync](https://github.com/ErikReider/SwayNotificationCenter) |
| Shell Prompt | [Oh-my-posh](https://ohmyposh.dev) |
| Shell | Zsh |
| Editor | [Neovim](https://neovim.io) |
| File Manager | Thunar / [Yazi](https://yazi-rs.github.io) |
| PDF Viewer | [Zathura](https://pwmt.org/projects/zathura) |
| Image Viewer | [imv](https://sr.ht/~exec64/imv) |

## ⌨️ Keybindings

| Key | Action |
|-----|--------|
| `Super + Return` | Terminal |
| `Super + Space` | App Launcher |
| `Super + E` | File Manager |
| `Super + B` | Browser |
| `Super + G` | Screen Record |
| `Super + S` | Screenshot |
| `Super + Y` | Yazi |
| `Super + Z` | Zathura |
| `Super + Shift + E` | Emoji picker |
| `Super + Escape` | Wlogout |

## 🚀 Install
```bash
git clone --depth=1 https://github.com/hoangnhtminh/dotfiles
cd dotfiles
cp -r .config/* ~/.config/
cp .zshrc ~/
cp .zprofile ~/
```
