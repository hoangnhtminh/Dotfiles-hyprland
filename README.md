## 🚀 Install
```bash
git clone --depth=1 https://github.com/hoangnhtminh/dotfiles
cd dotfiles
chmod +x install.sh
./install.sh
```

> ⚠️ Backup config before install

## ⌨️ Keybindings

| Key | Action |
|-----|--------|
| `Super + Return` | Terminal |
| `Super + Space` | Launcher |
| `Super + E` | File Manager |
| `Super + B` | Browser |
| `Super + G` | Record |
| `Super + S` | Screenshot |
| `Super + Y` | Yazi |
| `Super + Escape` | Wlogout |
